## Philosophy

-   Start small and adapt accordingly
-   Scalability
-   Pricing scales to 0 (Shouldn't be charged if low/no usage)
-   Easy to develop; Quick to configure

-   NextJS

-   Tailwind CSS

-   NextAuth for user authentication
-   Upstash for caching
-   Jest for testing

**Database**

-   MongoDB
-   Planetscale

**DevOps**

-   Github
-   Vercel

**CMS**

-   Contentful
